Comment installer ?
================
- Dezipper le fichier SPC5Studio.zip dans C:\SPC5Studio
- Installer le debugger à partir du fichier "ude-4-07-08-spc5-udestk.exe"
- Installer le cppchecker à partir du fichier  "cppcheck-1.77-x64-Setup.msi"
- Suivre les instruction du document "PLS_Debugger_Installation_driver_on_SPC560X_Discovery_board.pdf" pour activer le JTAG et la liaison série.
